# Mapa de recursos incorporados

La clase recoge ideas de los recursos LaTeX aportados, pero los reinterpreta como componentes propios y publicables. No se han incorporado documentos personales ni contenido privado.

| Recurso aportado | Integración en la plantilla |
|---|---|
| Preámbulos Pandoc/Eisvogel | Compatibilidad `Highlighting`, `Shaded`, tokens de sintaxis y macros `\tightlist`. |
| Plantillas de apuntes SQL | Entornos `ClaudeCode`, `ClaudeConsole`, `ClaudeOutput`, `ClaudeExercise` y `ClaudeSolution`. |
| Estilos `listings` para Java, C, C++, C#, VBA y SQL | Lenguajes definidos o preparados en `claudeeditorial.cls`. |
| `yReport` | Portada editorial, cabeceras, notas marginales, citas laterales, página de capítulo y banda vertical. |
| `yPrintImageToArea` | Macro `\ClaudeFitImage` para ajustar imágenes a un área mínima. |
| `yCards` | Entorno `ClaudeCard`, footer de card y raster de componentes. |
| `yBanner` | Macro `\ClaudeBanner` para separadores editoriales. |
| `infoBulle` y `marginInfoBulle` | Cajas semánticas y variantes marginales: info, tip, warning, check, question, critical. |
| `yFlatTable` | Tablas planas con `ClaudeTable`, filas alternas y cabeceras cálidas. |
| `tikz-uml` | Macros UML nativas `\ClaudeUMLClass`, `\ClaudeUMLInterface`, `\ClaudeUMLRelation`. |
| `tikz-er2` y crows-foot | Macros ER nativas, entidades, atributos, relaciones y cardinalidades `\ClaudeEROneToMany`. |
| Repositorio Open Design / Claude | Paleta cálida, disciplina tipográfica, anti-gradiente decorativo, acento sobrio y tono editorial. |

## Criterio legal y editorial

Los recursos originales sirven como referencia funcional. El repositorio resultante evita copiar manuales completos, contenido personal, imágenes privadas o textos con atribución obligatoria. La plantilla se publica con licencia sin requisito de atribución.
